# Remote Debugging Proxy #

> 远程调试代理服务器

## 使用方法 ##

把 conf.sample.json 复制为 conf.json，修改配置数据，键是需要处理的域名，值是指向本机的端口号。

在命令行中启动服务：

    npm install
    npm run start
